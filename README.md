# Curso: Aprenda Programação para seu Primeiro Estágio

Este é um curso completo e prático para iniciantes absolutos que querem entrar no mercado de tecnologia e conquistar seu primeiro estágio em programação.

## Sobre o Projeto

Este projeto contém um site completo de curso online, incluindo:

- **Página inicial** com apresentação do curso
- **Área do aluno** com 10 lições detalhadas
- **Página sobre** o instrutor e depoimentos
- **Página de preços** com plano único de R$ 48
- **Página de contato** com formulário
- **Design responsivo** e profissional

## Conteúdo do Curso

1. **Boas-vindas e Introdução**
2. **Como Funciona a Programação**

## Como Usar

1. Clone ou baixe os arquivos
2. Abra `index.html` no navegador
3. Navegue pelas páginas usando o menu
4. Na área do curso, clique nas lições na barra lateral
